## 摘要
机器人操作通常需要记忆：遮挡和状态变化可能导致决策时的观测出现感知混淆，使得动作选择在观测层面呈现非马尔可夫性，因为相同的观测可能源于不同的交互历史。大多数具身智能体通过语义压缩的轨迹和基于相似性的检索来实现记忆，这会丢弃用于消除歧义的细粒度感知线索，并可能返回感知上相似但与决策无关的情节。受人类情景记忆的启发，我们提出了 Chameleon，它通过写入基于几何的多模态 token 来保留消除歧义的上下文，并通过可微记忆栈产生目标导向的回忆。我们还引入了 Camo-Dataset，这是一个基于 UR5e 真实机器人的数据集，涵盖了感知混淆下的情景回忆、空间追踪和序列操作。在各项任务中，Chameleon 在感知易混淆的设置下，相较于强基准模型，一致地提高了决策可靠性和长程控制能力。

## Abstract
Robotic manipulation often requires memory: occlusion and state changes can make decision-time observations perceptually aliased, making action selection non-Markovian at the observation level because the same observation may arise from different interaction histories. Most embodied agents implement memory via semantically compressed traces and similarity-based retrieval, which discards disambiguating fine-grained perceptual cues and can return perceptually similar but decision-irrelevant episodes. Inspired by human episodic memory, we propose Chameleon, which writes geometry-grounded multimodal tokens to preserve disambiguating context and produces goal-directed recall through a differentiable memory stack. We also introduce Camo-Dataset, a real-robot UR5e dataset spanning episodic recall, spatial tracking, and sequential manipulation under perceptual aliasing. Across tasks, Chameleon consistently improves decision reliability and long-horizon control over strong baselines in perceptually confusable settings.

#### 1. 核心问题与整体含义（研究动机和背景）
在长程机器人操作中，**感知混淆（Perceptual Aliasing）**是一个核心瓶颈。由于遮挡、状态变化或任务的序列性质，机器人仅凭当前的观察往往无法做出正确决策（即非马尔可夫过程）。例如，在“三仙归洞”（Shell Game）游戏中，多个杯子外观完全相同，机器人必须依赖历史移动记录才能找到藏有目标的杯子。
现有的具身智能体记忆机制通常依赖于语义压缩（如将图像转为文字描述）或基于相似性的检索（RAG 模式），这会导致两个问题：
*   **细粒度信息丢失**：语义压缩丢弃了对决策至关重要的空间几何和视觉细节。
*   **检索干扰**：基于相似性的检索可能会召回视觉相似但决策无关的历史片段。
**Chameleon** 的研究动机是模仿人类的“情景记忆”机制，通过保留几何关联的细粒度信息和目标导向的检索，解决长程操作中的感知歧义问题。

#### 2. 论文提出的方法论
Chameleon 提出了一个受生物启发的“感知-记忆-策略”闭环架构：
*   **感知层（Perception）**：采用双摄像头（固定前视+手眼相机）。核心思想是利用机器人末端执行器（EE）作为**几何锚点**。通过对极几何（Epipolar Geometry）约束和几何偏置注意力机制，将多视图观测融合为具有视图一致性的 Patch Tokens，确保在写入记忆前保留空间位置信息。
*   **记忆层（Memory）**：模拟人类的内嗅皮层-海马体-前额叶（EC-HC-PFC）系统。
    *   **分层可微记忆栈**：耦合了“情景状态”（慢速更新，存储长程轨迹）和“工作状态”（快速更新，用于即时控制）。
    *   **时空锚点**：利用选择性状态空间模型（SSM/Mamba）在不同时间尺度上处理信息，减少干扰。
    *   **HoloHead（模式补全）**：引入一个潜在想象目标，训练记忆状态 $h_t$ 去预测未来的 2D/3D 航点。这迫使记忆状态不仅是压缩，而且必须具备预测未来任务进展的能力。
*   **策略层（Policy）**：基于**修正流匹配（Rectified Flow Matching）**生成末端执行器的未来轨迹。相比于单步预测，这种生成式轨迹建模提高了长程任务的稳定性。

#### 3. 实验设计
*   **数据集与场景**：引入了 **Camo-Dataset**，这是一个在真实 UR5e 机器人上采集的数据集，专门设计用于诱发感知混淆。包含三个任务：
    1.  **清理指定盘子**（情景回忆）：在多个外观相同的盘子中识别出之前被标记为“脏”的那一个。
    2.  **三仙归洞游戏**（空间追踪）：在杯子交换位置后，追踪藏有物体的杯子。
    3.  **添加多种调料**（序列操作）：按特定顺序使用勺子，需记忆已完成的步骤。
*   **Benchmark 与对比方法**：
    *   **基准模型**：Diffusion Policy（扩散策略）、Flow Matching（流匹配）、ACT（动作分块 Transformer）。
    *   **消融变体**：包括无记忆版本、基于检索的记忆库版本、纯 Mamba 版本、无几何分支版本等。

#### 4. 资源与算力
*   **硬件**：实验使用了单张 **NVIDIA GeForce RTX 4090 GPU**。
*   **训练时长**：每个任务训练约 **20,000 个步数（steps）**。
*   **推理性能**：端到端推理延迟平均为 **82 毫秒**，能够满足 UR5e 机器人的实时操作需求。

#### 5. 实验数量与充分性
*   **实验规模**：每个任务类别进行了 **36 组独立的真实机器人测试**（总计超过 100 场实机测试）。
*   **消融实验**：设计了 5 种不同的消融模型，涵盖了感知、记忆结构和训练目标等关键组件。

#### 6. 主要结论与发现
*   **记忆是长程任务的关键**：在感知混淆场景下，缺乏记忆的基准模型（如 Diffusion Policy）表现接近随机猜测，而 Chameleon 显著提升了决策可靠性。
*   **几何关联至关重要**：消融实验证明，如果没有基于末端执行器的几何偏置，空间追踪任务的成功率会大幅下降。
*   **HoloHead 的作用**：通过预测未来航点来约束记忆状态，能有效防止记忆坍缩，使机器人能够从部分线索中“补全”任务目标。
*   **性能提升**：在情景回忆任务中，Chameleon 的决策成功率达到了 100%，远超所有基准模型。

#### 7. 优点
*   **生物启发性强**：将神经科学中的情景记忆理论（模式分离与补全）成功转化为可工程化的可微架构。
*   **解决感知歧义**：通过几何锚点和多尺度 SSM 解决了机器人领域长期存在的非马尔可夫决策难题。
*   **实机验证**：所有实验均在真实物理机器人上完成，而非仅限于仿真环境，具有极高的实用价值。
